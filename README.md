# 开发

- 项目初始化

```
npm install
```
- 安装前切换到淘宝register

```
npm install nrm -g
nrm use taobao
```
- 项目启动

```
npm run dev
```
- 使用Mock启动项目

```
npm run dev:mock
```
- 打包

```
npm run build
```
- Lint with [ESLint](https://eslint.org/)

```
npm run lint
```
